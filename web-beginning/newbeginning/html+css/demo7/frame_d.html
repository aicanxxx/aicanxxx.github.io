西电新闻网讯（记者 吴华）8月30日下午，学校在南校区办公楼210报告厅召开2016年下半年工作会议，安排部署学校下半年重点工作。会上，党委书记陈治亚作了题为《加强三个一流建设，推动学校又好又快发展》的讲话，校长郑晓静院士代表领导班子作工作报告。会议由党委副书记龙建成主持。副校长杨银堂、李建东，党委副书记、纪委书记蒋舜浩、总会计师刘延平，校长助理高新波，全校博士生导师、科级及以上干部、直属单位负责人、各民主党派和侨联负责人、市级以上人大代表和政协委员等，共300余人参加了会议。‍
